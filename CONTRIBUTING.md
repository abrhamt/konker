# Contributing to Konker

We love contributions! Here's how to help.

## ---

## Code Style

- ESLint + Prettier
- Run `npm run lint` before commit

## ---

## Branching

- `main` → Production
- `develop` → Staging
- Feature: `feat/login-ui`

## ---


## Pull Requests

1. Fork & clone
2. Create branch
3. Commit with message: `feat: add login screen`
4. Open PR to `develop`
