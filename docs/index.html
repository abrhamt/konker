🃏 Konker

Ethiopian Rummy 41 - Real-time Multiplayer Card Game

About Konker

Konker is an authentic digital implementation of the traditional Ethiopian card game, Rummy 41 (Konker).

Play in real-time with friends, master strategic card play, and experience the classic game in a modern web environment.

React Node.js Socket.io MySQL

How to Play

Goal: Be the first to meld all cards with a minimum of 41 points.

Setup: Dealt 13 cards (dealer gets 14). Form sequences or groups.

Win Conditions: Standard Win or Doobie Win with special rules.

Turn Limit: 60 seconds per turn.

🎮 Game Features:

Real-time multiplayer • Server-validated moves • Authentic game rules • 60-second turn timers • Room codes for easy joining • Player statistics & history
