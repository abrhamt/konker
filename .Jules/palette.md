## 2024-03-20 - Adding ARIA attributes and Title to Icon-Only Elements\n**Learning:** Using `tsc -b` or running TS checks with default options on a legacy codebase with conversational artifacts can trigger widespread emitting of `.js` files if the `.tsconfig` lacks `noEmit` or isn`t configured correctly.\n**Action:** When validating syntax in a polluting environment, strictly use isolated execution `npx tsc --noEmit` and review `git status` closely to avoid committing generated JS files.
